# 《我的高考志愿模拟器》网页版开发计划

## 架构
- 单文件 HTML（含内嵌 CSS + JS），零外部依赖，可直接在浏览器打开
- 移动端优先（375px 宽度），PC 端居中留白
- 状态机驱动：一个 `currentScreen` 变量控制界面切换
- 游戏数据存在 `gameState` 对象中，方便后续移植 Unity
