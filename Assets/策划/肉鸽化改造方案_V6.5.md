# V6.5 肉鸽化改造：装备协同系统 + 学期重设计

## 一、装备协同系统（Synergy System）

### 1.1 核心理念

**流派不是选的，是"长"出来的。**

玩家不选职业/流派，而是通过装备搭配 + 事件选择，在游戏过程中自然形成build。这最像《杀戮尖塔》——你拿什么遗物就自然形成什么打法。

### 1.2 装备标签体系

每件装备自带1-2个**流派标签**：

| 标签 | 图标 | 核心效果 | 典型装备 |
|------|:--:|----------|---------|
| 📖 学习型 | 📖 | 智力/考试相关 | 学霸笔记、五年高考、错题本 |
| 🗣️ 社交型 | 🗣️ | 社交/信息/志愿策略 | 人脉名片、学生会徽、辩论手稿 |
| 🏃 健康型 | 🏃 | 健康/抗压/体力 | 运动手环、蛋白粉、跑鞋 |
| 💤 减压型 | 💤 | 压力管理/精神恢复 | 耳机、日记本、绿植 |
| 🎲 赌徒型 | 🎲 | 高方差/事件翻倍 | 幸运硬币、骰子、塔罗牌 |
| ⚖️ 均衡型 | ⚖️ | 全属性平滑增长 | 日程本、全能手环 |

### 1.3 协同规则（Synergy Rules）

**规则1：标签共鸣（Tag Resonance）**
装备2件同标签装备 → 触发共鸣效果

```
共鸣层数：
  2件同标签 → 共鸣Lv.1：该标签效果×1.3
  3件同标签 → 共鸣Lv.2：该标签效果×1.6 + 解锁专属事件池
  4件同标签 → 共鸣Lv.3：该标签效果×2.0 + 学期末额外奖励
  5件（全套） → 共鸣MAX：终极效果 + 隐藏成就
```

举例：装3件📖学习型装备 → 所有学习型装备效果×1.6，解锁"学习型专属事件池（刷题竞赛、夏冬营等）"

**规则2：槽位联动（Slot Combo）**
特定槽位组合触发额外效果：

| 组合 | 名称 | 效果 |
|------|------|------|
| 📖头 + 📖手 | **学神组合** | 智力额外+2 |
| 🗣️头 + 🗣️饰品 | **社牛组合** | 社交额外+2 |
| 🏃身 + 🏃脚 | **运动组合** | 健康额外+3 |
| 📖头 + 💤脚 | **开夜车组合** | 智力+2但压力+3/天 |
| 💤身 + 💤饰品 | **佛系组合** | 压力-2/天 |

**规则3：冲突标签（Tag Conflict）**
某些标签共存触发debuff（鼓励纯build）：

| 冲突 | 效果 |
|------|------|
| 📖×3 + 🎲×1 | "分心"：所有学习效果-10% |
| 🏃×3 + 💤×1 | "懈怠"：健康效果-15% |
| 🎲×3 + ⚖️×1 | "摇摆"：赌徒波动×0.5 |

**规则4：混搭自由（Mix & Match）**
冲突只在3+同标签时触发。2件混搭**无惩罚**，鼓励探索：

```
2📖 + 2🗣️ + 1💤 → 学习共鸣Lv.1 + 社交通鸣Lv.1 + 无冲突
→ "学霸社牛"混搭流 ✓
```

### 2.2 学期中：挂机+微决策流

```
时间轴（5个阶段，每阶段约1分钟）：

阶段1 (第1-21天)  ──→ 日常挂机 ──→ 微决策×3
阶段2 (第22-42天) ──→ 日常挂机 ──→ 微决策×3 + 可能期中事件
阶段3 (第43-63天) ──→ 日常挂机 ──→ 微决策×3 + 流派专属事件
阶段4 (第64-84天) ──→ 日常挂机 ──→ 微决策×3 + 关键抉择×1
阶段5 (第85-100天)──→ 日常挂机 ──→ 微决策×2 + 学期末事件

总计：约5分钟，15-20次微决策
```

**微决策事件频率提升：**

| 对比 | 旧设计 | 新设计 |
|------|--------|--------|
| 微决策间隔 | 每5-7天1次 | **每5-6天1次** |
| 5分钟学期决策数 | 约5-8次 | **约15-20次** |
| 流派专属事件 | 0 | **每学期3-5次** |
| 关键抉择 | 每学期1-2次 | **每学期2-3次** |
| 连锁事件 | 0 | **每学期1-2条** |
| 倒计时选择时间 | 无 | **5-10秒** |

### 4.2 连锁事件设计（核心亮点）

连锁事件 = 2-4个连续事件组成的故事线，增强沉浸感：

```
【竞赛之路】连锁 (学习型专属)
  事件1: "老师建议你参加奥赛" → [参加/婉拒]
  事件2: "竞赛集训开始" → [全力以赴/划水]
  事件3: "省级竞赛当天" → 根据前两次选择产生4种结局
  奖励: 紫装"竞赛金牌🎖️" 或 属性大幅提升

【校园风云】连锁 (社交型专属)
  事件1: "学生会改选开始报名" → [竞选主席/竞选部长/观望]
  事件2: "你的竞选演讲" → [准备演讲稿/即兴发挥/请人代写]
  事件3: "投票日" → 通过训练营时建立的NPC关系网决定胜负
  奖励: 紫装"学生会主席证" 或 NPC好感大幅提升

【运动会】连锁 (健康型专属)
  事件1: "运动会报名" → [参加/当啦啦队/逃课]
  事件2: "训练受伤了" → [坚持训练/休息养伤]
  事件3: "决赛日" → 连续QTE小游戏
  奖励: 紫装"运动健将勋章"

【早恋线】连锁 (特殊彩蛋)
  事件1: "收到了一封情书" → [接受/婉拒/无视]
  事件2: "被班主任发现了" → [承认/否认/求助家长]
  事件3: "选择" → 影响社交+压力+重要NPC关系
  奖励: 特殊天赋卡"少年心事" ± 效果
```

## 五、流派涌现示例

以下展示玩家通过**装备选择**自然形成不同流派的路径：

### 路径A：纯刷题流（📖×5，共鸣MAX）

```
装备选择: 学霸笔记(头) + 校服(身) + 五年高考(手) + 知识之靴(脚) + 状元符(饰品)
共鸣: 📖×5 = 共鸣MAX
效果: 智力增长+100%, 考试+12, 专属事件池(竞赛之路+刷题竞赛+复读营)
弱点: 社交增长-80%, 健康增长-50%, 压力+8/天 → 需要精准管理压力
学期末: 智力大涨但其他属性惨淡 → "小镇做题家"成就解锁
```

### 路径B：社牛流（🗣️×4+💤×1，共鸣Lv.3）

```
装备: 人脉名片(头) + 辩论队服(身) + 学生会徽(手) + 交际舞鞋(脚) + 塔罗牌(饰品)
共鸣: 🗣️×4 = 共鸣Lv.3
效果: 社交增长+160%, NPC好感+40%, 事件信息提前2天
终局: 社交属性极高 → 志愿填报策略信息全知 → "情报网"成就解锁
```

### 路径C：混搭赌狗流（🎲×2+📖×2+💤×1）

## 六、数值策划更新要点

### 6.1 共鸣加成公式

```
共鸣倍率:
  Lv.0 (0-1件同标签): ×1.0
  Lv.1 (2件同标签):  ×1.3
  Lv.2 (3件同标签):  ×1.6
  Lv.3 (4件同标签):  ×2.0
  MAX (5件同标签):   ×2.5

冲突惩罚:
  3+同标签(A) + 不同标签(B) = A共鸣-1级
  例: 3📖 + 1🎲 → 📖本应为Lv.2, 降为Lv.1
```

### 6.2 每日事件触发概率微调

```
原: 普通12% + 稀有4% + 史诗1.5% + 传说0.5% = 18%/天
新: 普通10% + 稀有3% + 史诗1.5% + 传说0.5% = 15%/天
调整原因: 现在多了流派专属事件(共鸣触发)和连锁事件，总事件密度实际上升
```
